CppNotes
========

My notes about C++.

##Requirements
g++ 5.2+, cmake 2.8+

## Build

``` bash
mkdir CppNotes-build
cd CppNotes-build
# Build type - Release or Debug
cmake .. -DCMAKE_BUILD_TYPE=Release
make
```
